Animations with p5.js

This project is an introductory assignment on using the Processing Library to create animations for Professor Kate Compton's CS 396: Generative Methods class. Creating the animations shown below involves learnin how to draw shapes with p5.js as well as learning how to use math and logic to make the animations seem seamless. The created animations here are intended to lay the foundation for creating more elaborate generative art, such as emergent particle systems.
